俄罗斯方块

请选择游戏模式

游戏规则

  • 方块堆积到顶部时游戏结束
  • 消除1行得10分，消除2行得30分，消除3行得60分，消除4行得100分
  • 双页面模式下按Tab键切换操控的面板

操作提示

  • ← 或 A 键：向左移动一格
  • → 或 D 键：向右移动一格
  • ↑ 或 W 键：旋转方块
  • ↓ 或 S 键：向下移动一格
  • 空格键：交换当前方块和下一个方块
  • Tab 键：双页面模式切换操控面板

单页面模式

分数: 0
操作提示：←/A左移 | →/D右移 | ↑/W旋转 | ↓/S下移 | 空格交换方块

双页面模式

左侧分数: 0 | 右侧分数: 0 | 总分: 0
操作提示：←/A左移 | →/D右移 | ↑/W旋转 | ↓/S下移 | 空格交换方块 | Tab切换面板（双页面模式）

双人对战模式

左侧玩家分数: 0 | 右侧玩家分数: 0 | 总分: 0
左侧玩家：W旋转 | A左移 | S下移 | D右移 | 空格交换方块
右侧玩家：↑旋转 | ←左移 | ↓下移 | →右移 | 空格交换方块
